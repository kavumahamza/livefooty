## Python + Django versions chosen and rationale

- **Python:** 3.13.13 (via `/usr/bin/python3.13`)
  - System Python is 3.14.4 but a 3.13 interpreter was available (`which python3.13` → `/usr/bin/python3.13`).
  - The venv at `backend/venv` was already initialised with Python 3.13.13.
  - Django 5.2 LTS officially targets Python 3.10–3.13, so 3.13 is the most compatible choice available.
  - Python 3.14 was skipped because Django 5.2 does not officially support it yet.

- **Django:** 5.2.15 (LTS release) — latest patch of the 5.2 LTS line, fully compatible with Python 3.13.
